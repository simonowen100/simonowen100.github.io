About this site:

This website is inspired by the Web 1.0/ early web 'Geocities' style of personal websites from the late 90's and early 00's. This aesthetic relied on crappy gifs, simple coding, and maximum personalisation—far beyond what's available with today's social media.
I use CodePen to draft my site and I host on GitHub Pages.
